Restaurant Supply Delivery Service Database

Tools: SQL

This project involved the design, implementation and documentation of an online system for a restaurant supply delivery service. Classical methodology guided the development of a relational database management system, using MySQL Server. An enhanced entity relationship diagram was constructed to parse the business use case and reflect its requirements, after which relational and database schema were created using various SQL statements based on this framework. This helped improve the data integrity and scalability to prevent corruption and accommodate future growth. For enhanced flexibility and usability, multiple views and stored procedures were implemented to allow system operators to access information and modify the database. The database system was integrated with a fully functional web-based application for user access.

Achievements

  • Improved data integrity and scalability for a delivery service to prevent corruption and accommodate growth, by parsing entity relationship requirements and developing database schema, using MySQL Server.
  • Enhanced flexibility and usability for system operators to access information and modify the database, by crafting multiple views and stored procedures for full functionality in an integrated web-based application.
