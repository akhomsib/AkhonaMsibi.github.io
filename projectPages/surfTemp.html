Climate Change: Surface Temperature Prediction
and Visualization

Tools: Python, d3.js

The aim of this project was to create an easily accessible platform that accurately and effectively communicates the reality of climate change to the public. This was with the objective to increase community awareness on the current situation, as well as constructing accurate predictive models to forecast future global temperatures. Several classical time-series models and novel deep learning techniques were utilized to predict future global temperatures.

Ultimately, a random forest model was evaluated based on its superior performance and followed modern research in predicting gradually increasing temperatures over time. These results were visualized in a user-friendly, web-based interface geared towards non-professionals, using d3.js programming to create an interactive 3D choropleth map to attain predictions for different regions around the world over several decades.

Achievements

  • Employed classical machine learning and novel deep learning time series models to forecast changes in global surface temperatures, applying Random Forest model that exceeded predictive accuracy by 25% over alternatives.
  • Crafted user-friendly web-based UI geared towards non-professionals, using d3.js to create an interactive 3D choropleth map to attain predictions for different regions around the world over several decades.
